# Projeto de Controle de Transações Bancárias

Este projeto foi desenvolvido durante os cursos da Alura:
- **TypeScript na prática: implemente um projeto completo com TypeScript e módulos**
- **TypeScript: aplicando orientação a objetos no Front-end**

## Sobre a Aplicação

Esta aplicação tem como objetivo simular o controle de transações bancárias, permitindo ao usuário:
- Visualizar o saldo atual da conta
- Registrar novas transações (depósitos e saques)
- Visualizar o extrato das transações realizadas

O projeto foi construído utilizando TypeScript, com foco em boas práticas de modularização e aplicação de conceitos de orientação a objetos no Front-end.
